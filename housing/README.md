# Lummi Nation Housing Authority Website

Complete website rebuild matching Concept B "Living Heritage" design style.

## Pages Built

1. **index.html** — Home page with hero, mission, 5 program cards, news preview, contact CTA
2. **programs.html** — All 5 programs with detailed descriptions and sub-programs
3. **about.html** — Mission, history, Director, full staff directory, Board of Commissioners
4. **forms.html** — All downloadable application forms with descriptions
5. **news.html** — Latest news and updates (5 articles)
6. **contact.html** — Contact information and contact form

## Design Features

- **CSS:** `../assets/css/style.css` (shared with health site)
- **Typography:** DM Serif Display (headings), DM Sans (body)
- **Color Palette:** Earth tones with river blue accents
- **Components:** Fade-in animations, mobile nav, scroll shadow, card layouts
- **Responsive:** Mobile-first design with breakpoints
